Simple Libraries Folder Index

Make sure you've got the latest, most up-to-date libraries. More info at Update Your Learn Folder.

Utility

___libsimpletools - Tools that simplify I/O, timing, memory, and communication

___libmstimer - Tracks milliseconds elapsed in another cog.

Text Devices

___libsimpletext - Tools for communicating with devices that transmit and/or receive characters and text strings. Includes half duplex serial drivers for Simple Terminal and other serial devices. Supports all device libraries in the Text Devices folder with stdio-like functions that feature smaller program size footprints.

___libserial - Functions for connecting to half duplex serial devices, for use with libsimpletext.

___libfdserial - Functions for connecting to full duplex serial devices, for use with libsimpletext.

Audio

___libwavplayer - Plays 16-bit, 32 ksps, mono .wav files (Propeller Activity Board/Propeller BOE).

Convert

___libadcDCpropAB - Propeller Activity Board A/D conversion for voltage measurements

___libadcACpropAB - Propeller Activity Board A/D conversion for signal measurements

___libdacctr - Propeller Activity Board D/A for setting voltages

Display

___libvgatext - Display text with a VGA Monitor.

___An LCD example lives in .../Documents/SimpleIDE/Learn/Examples/Devices/Display, library coming soon.

Motor

___libservo - Servo control library

___libservodiffdrive - Convenience functions for robot control with differential drive servos

___For DC motor control, see ...Documents/SimpleIDE/Learn/Examples/Devices/Motor/DC.

Sensor

___libcompass3d - For measurements with the 3-axis HMC5883L compass.

___libmx2125 - For measurements with the 2-axis mx2125 accelerometer.

___libping - For measurements with the Ping))) Ultrasonic Distance Sensor.
